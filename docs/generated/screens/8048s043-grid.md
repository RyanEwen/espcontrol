The home screen uses a **2-row x 3-column** grid, giving you **6 card slots**. Any home-screen card can be turned into a [Subpage](/features/subpages) folder containing up to 5 more cards.

Flexible card sizes are supported: Single, Tall, Wide, and Large.

| Capability | Value |

| Screen | 4.3 inches, 800 x 480, Landscape |
| Processor | ESP32-S3 |
| Built-in relays | No built-in relays |
| Image-based cards | Up to 6 simultaneous Image or Media Cover Art cards |
| Rotation support | Yes |
| Browser install slug | `sunton-esp32-s3-8048s043` |
| Ethernet option | No |
